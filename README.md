# Color Guesser

お題として表示される色（TARGET）と、スライダーで調整する色（YOURS）を見比べながら、同じ色を作り出す色当てゲームです。

## 特徴

- **マルチカラーモデル対応**: 馴染みのあるRGBに加え、HSV、HSL、人間の色覚認知に近いOKLAB、OKLCHといった様々な色空間（カラーモデル）に切り替えて遊ぶことができます。
- **2つのゲームモード**:
  - **フリープレイ**: 気軽に色を合わせる練習ができるモードです。
  - **5問チャレンジ**: 5回のラウンドでどれだけ正確に色を合わせられるかスコアを競い合うモードです。最後に結果をクリップボードにコピーしてシェアできます。
- **EASY MODE搭載**: 色空間の挙動がわからない場合でも、「EASY MODE（スライダーにグラデーションを表示）」にチェックを入れることで、各パラメーターがどう色に影響するかを視覚的に確認しながら直感的に操作できます。
